## 📁 Estrutura

- **📋 listas/** - Listas de exercícios (Lista A, Lista B, etc.)

## 🚀 Como compilar

```bash
# Compilar um arquivo
gcc nome_do_arquivo.c -o nome_do_programa

# Executar
./nome_do_programa
```
